# VAP 4.27 release

Read this page to learn what has changed in the Veracity Adapter for Power BI with the 4.27 release.

## What has changed
This release delivers UI updates to VAP to align the product with the DNV-wide visual standards. There are no functional changes to workflows or data handling.

## Benefits for users
- A more consistent look and feel with other DNV products.
- Improved readability and accessibility across the interface.
- A cleaner, more modern visual experience with fewer inconsistencies.

## UI changes

### Visual refresh and new corporate typography
VAP typography has been updated to the new DNV font. Text, headings and spacing have been adjusted to match the updated visual standards.

### Icons, spacing and component refinements
Several icons and separators have been updated and spacing/typography adjustments were applied across components (breadcrumbs, buttons, tables, inputs, tooltips).

Small UX polish has been applied to improve consistency and readability across screens.
