# VAP 4.27 release

Read this page to learn what has changed in the Veracity Adapter for Power BI with the 4.27 release.

This release includes frontend updates aligned with the latest Visual User Interface (VUI) standards. While there are no functional changes, the interface has been modernized for improved consistency, usability, and accessibility across DNV digital platforms.

## New features
There are no new user-facing features in this release.

## Changes in existing features

### Updated frontend interface with VUI alignment
The frontend has been refreshed to follow the latest DNV Visual User Interface (VUI) guidelines. This update ensures a consistent and modern look and feel across DNV applications, enhancing user experience through improved design, navigation, and responsiveness.

### Improved maintainability and design consistency
By adopting standardized UI components and styling, this release strengthens long-term maintainability and supports faster integration with future platform updates.
